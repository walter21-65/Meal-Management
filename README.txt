Meal Management - React + Firebase

Steps:
1. Create a Firebase project → Add Web App → Copy config
2. Paste the config inside src/firebaseConfig.js
3. Install Node.js → Run npm install → npm start
4. App runs on http://localhost:3000

To deploy on GitHub, use:
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/Meal-Management.git
git branch -M main
git push -u origin main
